A fun 3D basketball shooting game built with React and Three.js!

## Features
- 3D graphics with realistic physics
- Power charging shot system
- Score tracking and accuracy stats
- Smooth ball flight animation

## Play
Click and hold to charge, release to shoot!

## Run Locally
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
npm run preview
```

## Technologies
- React
- Three.js
- Vite
