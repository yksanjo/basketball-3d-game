# GitHub Repository Setup Guide

## Repository Description

Copy this description to your GitHub repository settings:

```
A fun and interactive 3D basketball shooting game built with React and Three.js. Experience realistic physics, smooth animations, and test your shooting skills with a power-charging system. Play now and see how many shots you can make!
```

## How to Add Topics

1. Go to your repository: https://github.com/yksanjo/basketball-3d-game
2. Click on the ⚙️ **Settings** tab (or the gear icon)
3. Scroll down to the **Topics** section
4. Click "Add topics"
5. Paste or type the topics above (one per line or separated by commas)
6. Click "Save changes"

## Repository Settings Recommendations

- **Description**: Use the description above
- **Website**: Add your deployed URL (if deployed)
- **Topics**: Add all the topics listed above
- **Social Preview**: GitHub will auto-generate from your README
